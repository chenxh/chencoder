## 演示工程和测试代码
## 工程模板功能
1. 生成archetype,执行命令:mvn archetype:create-from-project,会在target目录下面生成generated-sources目录，这个就是生成的 archetype
2. 安装到本地，cd进入generated-sourced/archetype目录，执行:mvn install
3. 创建maven工程时，就可以使用新的archetype创建
